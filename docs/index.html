audio_scripter

v0.0.8

A real-time scriptable audio effects plugin. Write per-sample DSP scripts, press Apply, hear the result — no compilation step, no DAW restart.

Download View on GitHub Language manual

Features

⚡

Per-sample execution

Scripts run once per audio sample at full sample-rate precision. No block overhead, no approximation.

🔄

Hot reload

Edit your script, press Apply. The engine swaps the program lock-free — audio never stops.

🎛️

8 automatable macros

p1–p8 are full DAW parameters. Automate them, MIDI-map them, modulate them.

🔧

DSP primitives

Built-in filters (lpf1, hp1, svf), delay lines, envelope followers, saturation, and more.

💾

Persistent state

Variables prefixed state_ survive across samples. Build oscillators, filters, and feedback with ordinary assignments.

📦

VST3 · AU · Standalone

macOS and Windows builds. AU for Logic, VST3 for everything else, or run standalone without a DAW.

Quick start

Every script sees inL, inR (input samples), outL, outR (output — default to input), sr (sample rate), t (time in seconds), and p1–p8 (macro knobs).

Minimal pass-through

# empty script — outL/outR default to inL/inR

Macro-driven soft saturation

# @p1: Drive
# p1 = 0.4
drive = 1.0 + p1 * 5.0;
outL = tanh(inL * drive);
outR = tanh(inR * drive);

Phase-accumulator oscillator (440 Hz)

state_phase = wrap(state_phase + 440.0 / sr, 0.0, 1.0);
outL = sin(6.2831853 * state_phase);
outR = outL;

User-defined function

fn softClip(x, drive) {
    return tanh(x * (1.0 + drive)) / (1.0 + drive);
}

outL = softClip(inL, p1 * 4.0);
outR = softClip(inR, p1 * 4.0);

Example scripts

SVF morphing filter

# @p1: Cutoff  @p2: Q  @p3: Morph  @p4: Wet
cut = 0.01 + p1 * 0.32;
q   = 0.25 + p2 * 1.4;
lpL = svf(inL, cut, q, 0.0, 0);
bpL = svf(inL, cut, q, 1.0, 2);
hpL = svf(inL, cut, q, 2.0, 4);
procL = mix(mix(lpL, bpL, p3), hpL, p3 * p3);
lpR = svf(inR, cut, q, 0.0, 1);
bpR = svf(inR, cut, q, 1.0, 3);
hpR = svf(inR, cut, q, 2.0, 5);
procR = mix(mix(lpR, bpR, p3), hpR, p3 * p3);
outL = mix(inL, procL, p4);
outR = mix(inR, procR, p4);

Micro-delay widener

# @p1: Time  @p2: Feedback  @p3: Width  @p4: Wet
base = 12.0 + p1 * 520.0;
fb   = p2 * 0.55;
spread = 1.0 + p3 * 0.8;

delInL = inL + state_fbL * fb;
delInR = inR + state_fbR * fb;
dL = delay(delInL, base, 0);
dR = delay(delInR, base * spread, 1);
state_fbL = sat(dL, 0.6 + p2 * 0.9, 0);
state_fbR = sat(dR, 0.6 + p2 * 0.9, 0);
outL = mix(inL, dL, p4);
outR = mix(inR, dR, p4);

Iterative wavefold

# @p1: Drive  @p2: Iterations  @p3: Wet
fn iterFold(x, drv, iters) {
    y = x * drv;
    for (k = 0.0; k < 12.0; 1.0) {
        if (ge(k, iters)) { break; }
        if (abs(y) < 0.02) { continue; }
        y = fold(y, -1.0, 1.0);
    }
    return y * 0.65;
}
drive = 1.0 + p1 * 7.0;
iters = 1.0 + p2 * 11.0;
outL = mix(inL, iterFold(inL, drive, iters), p3);
outR = mix(inR, iterFold(inR, drive, iters), p3);

Ping-pong delay

# @p1: Time  @p2: Feedback  @p3: Wet
dtime = (0.05 + p1 * 0.45) * sr;
fb = p2 * 0.7;
dL = delay(inL + state_pingR * fb, dtime, 0);
dR = delay(inR + state_pingL * fb, dtime, 1);
state_pingL = dL;
state_pingR = dR;
outL = mix(inL, dR, p3);
outR = mix(inR, dL, p3);

More scripts in the examples/ directory. Load any .ascr file directly from the plugin's Load button.

Language reference

For the full manual with tutorials and design patterns, see the Language Manual.

Function / syntax Description
Special variables
inL, inRInput samples (read-only), nominally −1.0 … 1.0
outL, outROutput samples — default to inL/inR each sample
srSample rate in Hz
tElapsed time in seconds
p1 … p8Macro knobs, 0.0 – 1.0, DAW-automatable
state_*Any variable whose name starts with state_ persists across samples
Control flow
if (c) { } else { }Conditional — non-zero is truthy
while (c) { }Loop while condition is non-zero
for (i=s; end) { }Legacy for: i steps by 1 while i < end
for (i=s; cond; step) { }Extended for: explicit condition and step expression
break; / continue;Exit or skip in the innermost loop
fn name(a, b) { return x; }User-defined function
Math
sin(x), cos(x), tan(x)Trigonometric (radians)
abs(x), sqrt(x)Absolute value, square root
exp(x), log(x)Exponential, natural log
tanh(x)Smooth clipper, output in (−1, 1)
pow(a, b)a to the power b
min(a, b), max(a, b)Smaller / larger of two values
Shaping
clamp(x, lo, hi)Hard-limit to [lo, hi] — also aliased as clip
mix(a, b, t)Linear interpolation: a + (b−a)·t
wrap(x, lo, hi)Cyclic wrap inside [lo, hi]
fold(x, lo, hi)Fold back at boundaries (wavefolder)
crush(x, steps)Quantise to N amplitude levels
smoothstep(e0, e1, x)Smooth 0→1 S-curve between e0 and e1
Oscillators & noise
noise(seed)Deterministic hash noise, output in (−1, 1)
pulse(hz, duty)Square wave — 1.0 while phase < duty, else 0.0
Filters (optional id selects independent state lane)
lpf1(x, coeff [, id])One-pole low-pass filter
hp1(x, coeff [, id])One-pole high-pass filter
bp1(x, hpC, lpC [, id])Band-pass: hp1 → lpf1 in series
svf(x, cut, q, mode [, id])State-variable filter: mode 0=LP, 1=BP, 2=HP
slew(target, speed [, id])Slew-rate limiter — max change per sample
env(x, atk, rel [, id])Envelope follower with asymmetric smoothing
Delay & saturation
delay(x, samples [, id])Delay line — 1–96000 samples per lane
sat(x, drive [, mode])Soft-clipper: mode 0=tanh, 1=atan, 2=cubic
Operators (arithmetic, comparison, logical, bitwise)
a + b, a - b, a * b, a / bArithmetic (division returns 0 when divisor ≈ 0)
a < b, a > b, a == b, a != bComparison — return 1.0 (true) or 0.0 (false)
!a, a && b, a || bLogical NOT, AND, OR — short-circuit, return 1.0/0.0
a & b, a | b, a ^ bBitwise AND, OR, XOR (operands cast to int32)
a << b, a >> bBit-shift left / arithmetic right
Comparison functions (return 1.0 / 0.0)
gt(a, b), lt(a, b)Greater-than, less-than
ge(a, b), le(a, b)Greater-or-equal, less-or-equal
select(cond, a, b)Returns a if cond ≠ 0, else b

Install & build

Pre-built binaries

Download the latest release from GitHub. Unzip and copy the bundle to your plugin folder, or run the installer script: 

./install.sh -b

Build from source

Requires CMake 3.22+ and a C++20 compiler. JUCE is fetched automatically. 

./scripts/build_release.sh \
  --config Release --package
